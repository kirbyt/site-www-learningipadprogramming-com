Learning iPad Programming, Second Edition

This book is for anyone wishing to learn iPad programming. The book is written for programmers new to iOS and iPad programming as well as experience iPhone programmers wishing to jump start their iPad programming skills. Readers will get hands on with iPad programming by building different iPad applications.
